BdSL Character Dataset

It is the first bangladeshi sign language character dataset. 1800 images of 36 classes of BdSL characters are included in this dataset. Every set of data is collected from different volunteers from different institutes near Dhaka, Bangladesh.

Eshara-Bochon

BdSL Digit Dataset

It is the first bangladeshi sign language digit dataset. 1000 images of 10 classes of BdSL digits are included in this dataset. Every set of data is collected from different volunteers from different institutes near Dhaka, Bangladesh.
